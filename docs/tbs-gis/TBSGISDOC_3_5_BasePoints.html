О механике базовых точек

В некоторых случаях при разработке информационных моделей может быть полезно задавать сдвиг координат. Это позволяет работать с малым порядком координат, что положительно влияет на производительность САПР. Механика базовых точек была широко распространена при работе в Autodesk Revit.

Из САПР, для которых имеются плагины TBS GIS, работа с базовыми точками обязательна для Renga (т.к. при удалении геометрии от "нуля" Renga уже более чем на 1 км начинаются графические дефекты, и комфорт работы с информационной моделью резко падает). Для CAD, Robur это опциональная возможность, как правило, работа там ведется сразу в абсолютных координатах.

В TBS GIS параметры базовой точки влияют на:

  • импорт векторных геоданных, включая специальные конфигурации для МГГТ, Росреестр XML;
  • импорт растровых геоданных;
  • импорт текстовых файлов;
  • экспорт векторных геоданных;
  • загрузка открытых данных;
  • экспорт триангуляции в растр (DEM);

Для CAD

Информация о базовой точке проекта, как и о системе координат, записывается в пользовательские свойства файла:

  • NCAD_PROPERTY_BASEPOINT_X - координата базовой точки X, метры;
  • NCAD_PROPERTY_BASEPOINT_Y - координата базовой точки Y, метры;
  • NCAD_PROPERTY_BASEPOINT_Z - координата базовой точки Z, метры;
  • NCAD_PROPERTY_BASEPOINT_ANGLE - угол поворота, против часовой стрелки, в градусах;

Для Robur

Информация о базовой точке проекта, как и о системе координат, записывается в пользовательские свойства файла (в настоящее время не доступны для редактирования вручную и, следовательно, для использования в работе с Robur).

Для Renga

Информация о базовой точке проекта, как и о системе координат, записывается в свойства Участка:

  • TBSGIS_ProjectBasePointX - координата базовой точки X, метры - значение свойства "дробное число";
  • TBSGIS_ProjectBasePointY - координата базовой точки Y, метры - значение свойства "дробное число";
  • TBSGIS_ProjectBasePointZ - координата базовой точки Z, метры - значение свойства "дробное число";
  • TBSGIS_ProjectBasePointAngleGrad - угол поворота, против часовой стрелки, в градусах - значение свойства "дробное число";
